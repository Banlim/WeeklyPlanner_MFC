# WeeklyPlanner_MFC_2018
## 화면구성

###메인화면 

  1) Left  - (사용자의 프로필 사진 설정, 목표 설정 - edit control{변경, 수정 버튼}
    
     > Profile Panel
     
       * Profile Image & Profile Message
      
     > Sound Panel
      
       * Play & Stop & Prev & Next
  
  2) Center
  
    > 체크박스 { Daily 시간, 할 일}
    
      * 할 일 추가 form 설정
      
    > 프로그레스바 {그날의 목표 달성치}
    
      
  3) Right
  
    > D-Day
        > D-day 설정 - 날짜의 의미 ex> 생일, 개학, 방학, 수능 등등
      
        * D-day 추가, 삭제, 수정
        
        > 모달대화상자 - 마감 날짜, 제목, 내용(메모) ( + 아이콘 )
        
          
  4) 이전 기록 보기
  
    > TodoList DB 이용
      
       * 특정 날짜의 데이터를 가져와 이전 일정 및 이전 달성률에 대해 보여준다.
